# Intermediary Work Product

Build these before major drafting so the motion has a stable factual and legal backbone.

## Required Tables

Create or update:

- `00_Intermediary Artifact Index.csv`
- Workspace Index
- Exhibit List
- RJN List
- Table of Objections
- Subpoena Request Details
- Replacement Table
- Needed Inputs
- TOC / argument outline
- TOA / authority table
- Exhibit citation table

## Section Artifacts

Save section drafts as numbered files in `Intermediary work`. Use workflow-order numbering, for example:

- `21_MTC Section 01 - Caption.md`
- `33_Sep Stmt Section 01 - Caption.md`
- `37_RJN Section 01 - Caption.md`

Do not create root preview copies or duplicate artifacts unless the user asks. Do not rename source files in `Exhibits`, `Context (not exhibit)`, or `Authorities` unless expressly instructed.

Every section artifact should include:

- draft text
- source/citation notes
- unresolved placeholders
- authority verification flags
- example-repurposing notes when a shell/example is used

## Table Of Objections

Columns:

- objecting party
- source document
- objection/argument
- page/line reference if available
- response theme
- likely use in MPA/Sep Stmt
- supporting facts/authorities

Consolidate repeated objections for MPA drafting, but preserve source detail and original wording needed for the Separate Statement.

## Replacement Table

Columns:

- exact sample text/search term
- replacement/current text
- affected document
- notes/placeholders

Use exact text from the current draft when the user wants search-and-replace guidance.

## TOA / Authority Table

Use only user-supplied or approved authorities. Do not introduce legal authorities from memory.

Columns:

- authority
- type
- proposition
- pincite
- source file
- draft section
- status/needs verification

Distinguish:

- example-shell authorities
- user-supplied authorities
- Rutter/practice-guide leads that require case/statute verification
- Lexis-verified additions

## Needed Inputs

Columns:

- missing item
- why needed
- affected document/section
- priority
- suggested source

## Saving Tables

When the user asks for saved intermediary work, save tables in `Intermediary work`, usually as a spreadsheet named:

```text
Intermediary Work Tables.xlsx
```
