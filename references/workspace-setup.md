# Workspace Setup

Use this to organize subpoena MTC workspaces.

## Folder Structure

Expected folders:

- `Exhibits`
- `Context (not exhibit)`
- `Authorities`
- `Intermediary work`

Main work product stays in the workspace root unless the user directs otherwise.

## Folder Roles

`Exhibits`:

- subpoenas
- proofs of service
- objections
- hold letters
- meet-and-confer letters
- tracing records
- authentication materials
- court orders or transcripts intended for RJN/exhibit use

`Context (not exhibit)`:

- operative pleadings used for context
- prior example motions
- prior oppositions
- background documents not currently intended as exhibits

`Authorities`:

- user-supplied cases
- statutes/rules/practice guides
- research memoranda
- legal authorities approved by the user

`Intermediary work`:

- OCR output
- tables
- outlines
- draft declarations for review
- change logs
- needed-input lists

## Document Index

Create or update a document index with:

- folder
- filename
- document type
- date
- parties/entities
- page count/OCR status
- role/purpose
- likely exhibit/RJN use
- notes/gaps

## Main Work Product

Root-level drafts usually include:

- Motion / MPA
- Separate Statement
- RJN
- Declaration
- Proposed Order

Do not move root-level work product unless the user directs it.
