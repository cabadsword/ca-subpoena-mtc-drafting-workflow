# Live Draft Change Log

Maintain a change log in `Intermediary work` when confirmed sections are inserted into Word drafts.

Recommended filename:

```text
Live Draft Change Log.md
```

## Entry Format

```markdown
## <date/time>

- Document: <filename>
- Section: <section title>
- Action: inserted / replaced / revised
- Confirmed by user: yes
- Source: chat proposal / user text / sample repurposing
- Remaining placeholders:
  - <placeholder or none>
- Notes:
  - <brief note>
```

## Placeholder Tracking

Track unresolved items such as:

- declaration paragraph numbers
- exhibit numbers
- RJN numbers
- hearing details
- sanction amount
- page/line objection cites
- authority pincites
- missing source documents

If no placeholders remain, say `none`.

## Change Log Rule

Each confirmed Word update should have a same-turn change-log entry unless the user directs otherwise.
