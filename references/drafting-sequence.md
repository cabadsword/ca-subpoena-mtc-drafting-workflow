# Drafting Sequence

Use this sequence unless the user directs otherwise. Start from any supplied example shell and closely repurpose its structure before tailoring.

## Motion / MPA

1. Caption
2. Notice
3. Introduction
4. Summary of Facts
5. Court authority to compel subpoena compliance
6. Relevance / need
7. Particularity / narrow tailoring / overbreadth
8. Privacy / confidentiality / protective order
9. Privilege / work product / trade secret / proprietary information
10. Burden
11. Meet and confer
12. Sanctions
13. Conclusion

Before drafting each issue, decide whether it belongs as a standalone section or a subsection under the main subpoena-compliance argument. Update the live TOC / argument outline so numbering stays coherent.

For each section:

- review current draft and sample language
- closely repurpose the example's structure and phrasing
- tailor parties, facts, dates, citations, headings, and relief
- save a numbered section artifact in `Intermediary work`
- wait for confirmation before updating any Word draft
- record confirmed updates and unresolved placeholders in the live change log

## Declaration

Typical order:

1. declarant identity and role
2. case context from pleadings if appropriate
3. factual tracing narrative
4. subpoena issuance and service
5. objections and production hold
6. meet-and-confer history
7. exhibit authentication
8. sanctions/time spent if needed

Use exhibit numbers consistently with the current exhibit list.

## Separate Statement

Before drafting, decide whether the subpoena categories are one request with internal categories or multiple separate requests. Use the user's direction if provided.

Typical order:

1. caption
2. introductory paragraph
3. full subpoena request text
4. original objection text, not a summary, with page:line citations where available
5. deponent response/hold position if separate from the objector
6. consolidated good-cause statement mirroring the MPA

Separate Statement argument should be shorter and more consolidated than the MPA. It should mirror the MPA structure, preserve key citations, and avoid re-briefing the entire motion.

## RJN

Use RJN only for proper judicially noticeable materials, typically court orders, minute orders, transcripts, filed pleadings, recorded public records, Secretary of State records, and similar court/public records.

Closely mirror the example RJN format. Tie the caption `RE:` line to the final RJN exhibit numbers. Request notice only for the existence, contents, recordation, filing, or legal effect of the record, not for the truth of disputed factual assertions unless legally proper.
